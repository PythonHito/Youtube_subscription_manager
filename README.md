# Abonnements-youtube

## How does it run
1. Recover your subscriprion files in youtube
2. Put this file in the directory
3. Run the script:
``` python3 youtube.py ```
4. If you want to add a sub. Append the id channel and the name channel in ` sub.swy`
    - like this: ```channel_id	name ```

## Requirements
- Python 3
